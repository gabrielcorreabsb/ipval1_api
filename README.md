# API Gerenciamento de Loja

API REST desenvolvida para gerenciamento de loja, utilizando Spring Boot.

## 🚀 Tecnologias Utilizadas

* Java 17
* Spring Boot
* Maven
* Spring Security
* Spring MVC
* Spring Data JPA
* CSS
* HTML
* postgreeSQL


## 📋 Pré-requisitos

* Java JDK 21
* Maven
* IDE de sua preferência (Eclipse, IntelliJ, VS Code)
* Git
